# BCI 技术全景研究报告

> 研究日期: 2026-06-13  
> 研究范围: 侵入式 vs 非侵入式脑机接口的完整技术栈对比  
> 目标: 确定最适合的技术路径和硬件方案

---

## 一、BCI 完整硬件栈

一个 BCI 系统从大脑到应用，经过 7 层硬件栈：

```
┌─────────────────────────────────────────────────────┐
│  Layer 7: 应用层 (Application)                        │
│  光标控制 / 语音合成 / 癫痫控制 / 专注力训练          │
├─────────────────────────────────────────────────────┤
│  Layer 6: 算法层 (Decoding / AI)                      │
│  Transformer / RNN / CSP+LDA / 基础模型              │
├─────────────────────────────────────────────────────┤
│  Layer 5: 处理器层 (Compute)                          │
│  片上 DSP / FPGA / GPU / 边缘 AI                     │
├─────────────────────────────────────────────────────┤
│  Layer 4: 模数转换层 (ADC)                            │
│  24-bit ADC, 采样率 250Hz ~ 30kHz                    │
├─────────────────────────────────────────────────────┤
│  Layer 3: 放大层 (Amplifier)                          │
│  仪表放大器, 增益 1000x-100000x, 输入阻抗 >1GΩ       │
├─────────────────────────────────────────────────────┤
│  Layer 2: 电极层 (Electrode)                          │
│  侵入: 微电极阵列 / 非侵入: 头皮电极                  │
├─────────────────────────────────────────────────────┤
│  Layer 1: 生物界面 (Bio-Interface)                    │
│  侵入: 脑组织接触 / 非侵入: 头皮-颅骨-脑膜           │
└─────────────────────────────────────────────────────┘
```

## 二、逐层技术对比

### Layer 1: 生物界面

| 参数 | 侵入式 (Intracortical) | 侵入式 (ECoG) | 微创 (Endovascular) | 非侵入式 (EEG) |
|---|---|---|---|---|
| **代表** | Neuralink N1, Blackrock Utah | Precision Layer 7 | Synchron Stentrode | OpenBCI, Emotiv |
| **电极位置** | 插入皮层 1-2mm | 放在皮层表面 | 血管内壁 | 头皮外 |
| **信号衰减** | 无衰减 | 轻微衰减 | 中等衰减 | 严重衰减 (颅骨) |
| **空间分辨率** | 单神经元 (10μm) | 局部场电位 (100μm) | 皮层区域 (1mm) | 大范围 (1cm+) |
| **信号类型** | Spike, MUA | ECoG, 高频γ | LFP, 高频γ | 头皮电位 (μV) |
| **手术风险** | 高 (开颅) | 中 (微创开颅) | 低 (导管) | 无 |
| **可逆性** | 几乎不可逆 | 可逆 | 可逆 | 完全可逆 |
| **长期稳定性** | 胶质疤痕问题 | 较好 | 多年稳定 | 无退化问题 |

**关键发现**: 非侵入式唯一的劣势是颅骨对信号的衰减。颅骨使 EEG 信号幅度衰减约 100 倍，空间分辨率降低约 100 倍。但这是物理限制，不是技术限制。

---

### Layer 2: 电极层

#### 侵入式电极

| 技术 | 通道数 | 材料 | 阻抗 | 寿命 | 代表公司 |
|---|---|---|---|---|---|
| **Utah Array** | 10×10=100 | 硅 | 100kΩ-1MΩ | 2-5年 | Blackrock |
| **Michigan Probe** | 32-128 | 硅 | 50kΩ-500kΩ | 1-3年 | 学术 |
| **Neuralink Thread** | 64线×16点=1024 | 聚酰亚胺 | 50kΩ | 目标>5年 | Neuralink |
| **Precision Layer 7** | 1024 | 薄膜金 | <10kΩ | 目标>5年 | Precision |
| **石墨烯电极** | 研发中 | 石墨烯 | <1kΩ | 目标>10年 | Inbrain |

#### 非侵入式电极

| 技术 | 通道数 | 材料 | 阻抗 | 信号质量 | 代表产品 |
|---|---|---|---|---|---|
| **湿电极 (Ag/AgCl)** | 32-256 | 银/氯化银 | <10kΩ | 最好 | Brain Products |
| **干电极** | 8-32 | 金属/导电聚合物 | 50-200kΩ | 中等 | Emotiv EPOC X |
| **柔性干电极** | 8-64 | 纳米织物 | 20-100kΩ | 较好 | Kernel Flow (52ch) |
| **fNIRS 光学** | 8-52 | LED+光电二极管 | N/A | 血流信号 | Kernel Flow |
| **TGAM 单通道** | 1 | 干电极 | >100kΩ | 最低 | NeuroSky |

**关键发现**: 非侵入式电极通道数正在快速追赶。Kernel Flow 已做到 52 通道 fNIRS，Emotiv MN8 做到 8 通道 EEG 集成到耳机形态。通道数差距在缩小，但信号质量差距仍然巨大。

---

### Layer 3: 放大层

| 参数 | 侵入式 | 非侵入式 |
|---|---|---|
| **输入阻抗** | >1GΩ | >100MΩ |
| **增益** | 1000x-10000x | 10000x-100000x |
| **CMRR** | >100dB | >110dB |
| **输入噪声** | <1μV rms | <0.5μV rms |
| **带宽** | 0.1Hz-20kHz | 0.1Hz-100Hz |
| **代表芯片** | Intan RHD2164, Blackrock CerePlex | ADS1299 (TI), AFE44xx |

**关键发现**: 放大器层侵入式和非侵入式用的是**同一类芯片**。TI ADS1299 同时被侵入式和非侵入式系统使用。这一层没有本质区别，区别在于前端电极阻抗不同导致对放大器要求不同。

---

### Layer 4: 模数转换层 (ADC)

| 参数 | 侵入式 | 非侵入式 |
|---|---|---|
| **位深** | 16-bit | 24-bit |
| **采样率** | 20-30kHz/通道 | 250-1000Hz/通道 |
| **数据带宽** | 200Mbps (Neuralink) | <10Mbps |
| **代表芯片** | 定制 ASIC | ADS1299 (24-bit, 250Hz) |
| **TGAM** | N/A | 8-bit, 512Hz (内置) |

**关键发现**: 侵入式需要定制 ASIC 处理 1024 通道 × 20kHz = 20MSPS 的吞吐量。非侵入式用现成芯片就够了。这一层非侵入式反而更简单、更便宜。

---

### Layer 5: 处理器层

| 参数 | 侵入式 | 非侵入式 |
|---|---|---|
| **形态** | 植入芯片 (Neuralink N1) | 外置设备 (手机/电脑) |
| **功耗限制** | <10mW (植入) | 无限制 |
| **算力** | 极低 (只做压缩+BLE) | 高 (GPU/CPU) |
| **代表** | Neuralink N1 ASIC | 手机 + 云端 |
| **AI 推理** | 无 (原始数据传出) | 可做实时 AI |

**关键发现**: 非侵入式的算力优势是碾压级的。植入芯片只能做信号压缩和无线传输，所有解码/AI 都在外部设备上完成。非侵入式可以直接在手机上跑深度学习。

---

### Layer 6: 算法层

| 方法 | 侵入式效果 | 非侵入式效果 | 通用性 |
|---|---|---|---|
| **FBCSP + LDA** | 好 (70-85%) | 一般 (55-70%) | ✅ 通用 |
| **EEGNet** | 好 (80-90%) | 中等 (65-80%) | ✅ 通用 |
| **Transformer** | 很好 (90-95%) | 好 (75-85%) | ✅ 通用 |
| **基础模型** | 最好 (95%+) | 好 (80-90%) | ✅ 通用 |
| **语音解码** | 78 wpm (ECoG) | 不可行 | ❌ 侵入式独有 |
| **手指级精细控制** | 可行 | 不可行 | ❌ 侵入式独有 |
| **光标控制** | 8+ bits/s | 1-2 bits/s | ✅ 通用但差距大 |
| **专注/放松检测** | 好 | 好 (TGAM直出) | ✅ 几乎无差距 |
| **P300/SSVEP** | 好 | 好 | ✅ 几乎无差距 |

**关键发现**: 算法层侵入式和非侵入式**用同一套方法**。差距不在算法，在输入信号质量。但某些应用（专注度、P300、SSVEP）在非侵入式下效果几乎和侵入式一样好。

---

### Layer 7: 应用层

| 应用 | 侵入式 | 非侵入式 | 市场大小 |
|---|---|---|---|
| **瘫痪者通信** | ✅ 唯一方案 | ❌ 带宽不够 | 小 (50万人) |
| **语音恢复** | ✅ ECoG 78wpm | ❌ 不可行 | 小 |
| **癫痫控制** | ✅ NeuroPace 4000+ | ⚠️ 监测可以 | 中 (600万人) |
| **机器人控制** | ✅ 精细控制 | ⚠️ 粗糙控制 | 中 |
| **专注力训练** | ✅ | ✅ 效果相当 | **大 (几亿人)** |
| **冥想/放松** | ✅ | ✅ 效果相当 | **大 (几亿人)** |
| **助眠** | ✅ | ✅ tDCS 有效 | **大 (几亿人)** |
| **游戏输入** | ⚠️ 太远 | ✅ SSVEP/EMG | 中 |
| **AR/VR 输入** | ❌ 太远 | ✅ EMG 手腕 | **大** |

## 三、写入路径 (Neuromodulation) 对比

BCI 不只是"读"，还有"写"——向大脑输入信号。

| 写入技术 | 侵入/非侵入 | 空间精度 | 深度 | 机制 | 代表 |
|---|---|---|---|---|---|
| **DBS** | 侵入 | 1mm | 深部 | 电刺激 | Medtronic Percept |
| **皮层电刺激** | 侵入 | 1mm | 皮层 | 电刺激 | Neuralink (双向) |
| **tDCS** | 非侵入 | 10mm+ | 浅皮层 | 直流电 | Flow Neuroscience |
| **tACS** | 非侵入 | 10mm+ | 浅皮层 | 交流电 | 学术 |
| **tFUS** | 非侵入 | **2mm** | **可达深部** | 超声 | Grey Matter |
| **TMS** | 非侵入 | 5mm | 浅皮层 | 磁场 | Magstim |
| **VNS** | 侵入 | N/A | 迷走神经 | 电刺激 | NeuroPace |

**关键发现**: tFUS 是非侵入式写入中**唯一**达到毫米级精度且能到达深部脑区的方案。177 个临床试验已注册 (2025)，85% 在近 5 年启动。

## 四、闭环系统对比

| 系统 | 读 | 写 | 闭环 | 代表 | 状态 |
|---|---|---|---|---|---|
| **NeuroPace RNS** | 颅内 EEG | 电刺激 | ✅ 实时 | 癫痫控制 | **FDA 已批准, 4000+ 患者** |
| **Neuralink** | 1024ch spike | 电刺激 (规划中) | ⚠️ 规划中 | 运动控制 | 临床试验 |
| **Synchron** | 16ch LFP | 无 | ❌ 单向 | 运动控制 | 临床试验 |
| **Flow Neuroscience** | 无 | tDCS | ❌ 开环 | 抗抑郁 | **已上市** |
| **Muse** | EEG | 声音反馈 | ⚠️ 间接 | 冥想 | **已上市, 200万用户** |

**关键发现**: 真正闭环（读+写+实时反馈）的系统只有 NeuroPace。所有非侵入式消费产品都是单向的（只读或只写）。**闭环非侵入式 BCI 是一个空白市场。**

## 六、组合方案：哪些组件可以跨层级复用

### 可复用的组件

| 组件 | 侵入式用 | 非侵入式用 | 可复用 |
|---|---|---|---|
| **FBCSP 特征提取** | ✅ | ✅ | ✅ 完全复用 |
| **EEGNet / Transformer 解码器** | ✅ | ✅ | ✅ 完全复用 |
| **伪迹去除 (ICA/小波)** | ✅ | ✅ | ✅ 完全复用 |
| **SSVEP 检测** | ✅ | ✅ | ✅ 完全复用 |
| **P300 检测** | ✅ | ✅ | ✅ 完全复用 |
| **BLE 无线传输** | ✅ | ✅ | ✅ 完全复用 |
| **ADS1299 前端** | ✅ | ✅ | ✅ 同一芯片 |
| **电极设计** | ❌ 完全不同 | ❌ 完全不同 | ❌ 不可复用 |
| **植入封装** | ✅ | ❌ | ❌ 不可复用 |
| **手术机器人** | ✅ | ❌ | ❌ 不可复用 |

### 最优组合策略

```
非侵入式硬件 (EEG 头环 + tDCS 电极)
        +
通用算法层 (FBCSP / EEGNet / Transformer)
        +
闭环控制器 (EEG 状态 → tDCS 强度)
        +
消费级应用 (专注力 / 助眠 / 冥想)
        =
一个有市场、有壁垒的产品
```

## 八、核心结论

### 已验证的事实

1. **NeuroPace 是唯一成功的闭环 BCI 公司** — 4000+ 患者，FDA 批准，有收入
2. **所有消费级 BCI 产品都是单向的** — 只读 (Muse/BrainCo) 或只写 (Flow)
3. **闭环非侵入式 BCI 是空白市场** — 没有人在做 EEG+tDCS 实时闭环消费品
4. **tFUS 是最有潜力的非侵入式写入** — 毫米级精度，177 个临床试验，但硬件不成熟
5. **算法层侵入式和非侵入式通用** — 我们的 FBCSP/CNN 代码可以直接复用
6. **专注力/助眠/冥想市场巨大** — 不需要医疗审批

### 推荐路径

```
Phase 1: EEG 头环 + tDCS 闭环原型 (3个月)
  → 验证闭环可行性
  → 目标: 桌面 demo

Phase 2: 移动端 App + 刺激方案库 (3个月)
  → 5 个预设方案 (专注/放松/助眠/冥想/恢复)
  → 目标: 10 人内测

Phase 3: 量产 + 众筹 (3个月)
  → BOM ¥112, 零售 ¥399
  → 目标: 100 台

Phase 4: 数据 + AI 个性化 (3个月)
  → 用户 EEG + 刺激配对数据
  → 个性化闭环算法
  → 目标: 500+ 用户
```
