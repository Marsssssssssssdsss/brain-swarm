> **注意**：立创的 ADS1299IPAGR 是 **编带包装**（R后缀表示编带），和 TQFP-64 的芯片完全一样，只是出厂时是编带给贴片机用的。你自己手工焊同一个芯片，没问题。

## 购买顺序建议

1. **第一步**：立创一次把芯片+阻容买完（运费 ¥10-15，隔天到）
2. **第二步**：淘宝买弹簧针、电池、TP4056、屏蔽线（运费 ¥5-8，2-3天到）
3. **第三步**：3D打印外壳（可以同时发打印店，2-3天）
4. **第四步**：嘉立创打 PCB（发 Gerber 文件，3-4天做好 + 2天快递）

**等全部零件到齐了开始焊接**，不要分批到一批焊一批。
