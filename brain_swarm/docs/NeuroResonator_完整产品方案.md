## 2. 三条路线分析

### 路线 1：非侵入式（头皮可穿戴）

```
EEG 干电极 → 头皮 → 颅骨 → 硬脑膜 → 蛛网膜 → 脑
                            ↑—— 信号衰减 ~10x ——↑
tDCS/tACS: 电流穿越 ~15mm 组织到达皮层，空间分辨率 ~10cm²
```

| 维度 | 评估 |
|------|------|
| 信号质量 | 10-20μV，SNR ~-20dB，需叠加/滤波提取 |
| 准确率 | ~70%（专注/放松足够），冥想类 App 用户本来就接受"大概对" |
| 上市时间 | **6 个月**（PCB → 固件 → App → 众测 → 小批量） |
| 监管 | 夏威夷私人岛屿 = 零监管，直接以"健康辅助设备"出货 |
| 总开发成本 | **~$18,500**（到 MVP） / **~$80,000**（到量产） |
| 售价 | $249（3× BOM $81） |
| 历史类比 | **VR 开发者套件** — 用现成组件快速搭建，验证市场后迭代升级 |

**关键优势**：你今天就可以做出原型。2026 年 7 月就能发货。

### 路线 2：微侵入式（血管内/硬膜下）

```
                             ┌→ 硬膜下电极 （开颅小孔，电极放在硬脑膜上）
导管 → 颈内静脉 → 矢状窦 ──┤→ Stentrode （支架电极，经血管植入）
                         └→ 脑表面阵列 （柔性贴片覆盖感觉/运动皮层）
```

| 维度 | 评估 |
|------|------|
| 信号质量 | 200-500μV，SNR ~0dB，可见清晰皮层节律 |
| 准确率 | ~90-95%，可做精细运动解码 |
| 上市时间 | **2-3 年**（需动物实验→临床→手术团队合作） |
| 监管 | FDA/CE 至少 Class II-III，需 IDE 审批 |
| 总开发成本 | **$2-5M**（动物实验 + 临床试验 + 手术设备） |
| 参考公司 | Synchron ($100M+ 融资，8 年才到人体试验) |
| 历史类比 | **Lattice 系统第一版** — 大量 COTS+现有技术拼出 MVP，迭代升级 |

**关键问题**：Synchron 从 2016 到 2024 才 FDA 获批人体试验。必须慢——但一旦做成，没有消费级公司能碰。

### 路线 3：侵入式（Neuralink 级）

```
微丝阵列 ↓ 穿透脑实质 ↓ 刺入皮层 ↓ 记录单个神经元放电
    ↓              ↓              ↓
1024 通道     免疫反应      神经外科手术
    ↓              ↓              ↓
需要动物实验→人体试验→监管→10年
```

| 维度 | 评估 |
|------|------|
| 信号质量 | 单神经元尖峰信号，理论带宽无限 |
| 上市时间 | **5-10 年** |
| 所需资金 | **$100M+**（Neuralink 烧了 $1.29B） |
| 实用性 | 即使有私人岛屿，CE/FDA 管的不是你在哪做实验——是产品能不能卖进市场 |

**侵入式路线不适合早期创业公司**。Neuralink 烧了 10 年 $1.29B 还没量产。一家国防级硬件公司 5 年做到 $5B 收入——方向不同。

## 3. 选定路线：混合路线

```
Phase 1 (2026 Q4)     →  Phase 2 (2027 Q2)     →  Phase 3 (2027 Q4+)
────────────────────────────────────────────────────────────────────
  非侵入式              →  扩展非侵入式           →  微侵入式接口
  $249, BOM $81       →  $449, BOM $200         →  $999+, BOM $500+

  ✅ 4ch 干电极 EEG     →  ✅ 16ch EEG             →  ✅ 硬膜下/血管内电极
  ✅ 7 种脑状态检测      →  ✅ 主动干电极 CNT-PDMS    →  ✅ 多通道刺激 (4×)
  ✅ tDCS + tACS 反馈   →  ✅ tFUS 超声聚焦          →  ✅ 更高精度 ADC
  ✅ 手机 App           →  ✅ 个性化 AI 模型          →  ✅ 多模态融合
  ✅ AI 云分析基础版     →  ✅ AI 教练 + 预测          →  ✅ 医疗级/国防级

  硬件价值: 3合1设备     →  硬件价值: 不存在的设备     →  硬件价值: 同类 $10K+
  AI 价值: 通用模型      →  AI 价值: 私有模型           →  AI 价值: 脑数据 API
```

**为什么非侵入式先行？**

已验证的硬件迭代逻辑：
1. 先用现成组件搭出第一版 MVP → 验证产品的核心价值
2. 早期出货收集真实用户数据 → 有反馈、有迭代方向
3. 用数据做第二代 → 比第一代好 10 倍

我们同理：
1. 先用 COTS 芯片搭出 Phase 1 → 验证闭合环路 EEG+tDCS 能行
2. 收集真实用户的脑数据 → 训练 AI v2
3. 用数据做 Phase 2 / 3 → 比 Phase 1 好 10 倍

**为什么 PCB 预留？**

不走"先卖 100 台 Phase 1，再把 100 台全扔掉重新设计 Phase 2"的路线。PCB 上物理存在升级焊盘——用户买了 Phase 1，Phase 2 时寄回来升级（换焊 ADS1299 #2 + tFUS 模块），环保 + 用户留存。

## 4. 市场分析

### 市场空白确认

| 产品 | 读 EEG | 写 (tDCS/tACS) | 闭合环路 | 价格 |
|------|--------|---------------|---------|------|
| **NeuroResonator** | **✅ 4ch** | **✅ tDCS+tACS** | **✅ 实时闭环** | **$249** |
| Muse 2 | ✅ 4ch | ❌ | ❌ | $249 |
| Flow FL-100 | ❌ | ✅ tDCS | ❌ | $459 |
| OpenBCI Cyton | ✅ 8ch | 需外挂 | ❌ | $349+$249 |
| LIFTiD | ❌ | ✅ tDCS | ❌ | $159 |
| NeuroSky | ✅ 1ch | ❌ | ❌ | $199 |
| Emotiv EPOC X | ✅ 14ch | ❌ | ❌ | $849 |

**结论**：没有竞品。NeuroResonator 是市场上第一个"读→分析→写"合一的产品。

### 为什么没人做？

1. **EEG 噪声太大** → 2010-2020 年的信号处理做不到 70% 准确率 → 2025 年 TinyML 可以了
2. **tDCS 效果微妙** → 单次使用效果弱（~0.3 Cohen's d）→ 长期持续使用效果好得多（神经可塑性重塑需要时间）
3. **跨学科难度** → 硬件 + 信号处理 + AI + 消费电子 + 神经科学 → AI 辅助开发让门槛降低了

## 6. AI 层设计

### 6.1 设备端 AI（TFLite Micro, ESP32-S3）

```
输入: 4ch × 256 samples (1s @250Hz) → FFT → 20 band powers

设备端模型架构:
  ┌─ Conv1D(8, kernel=16) ── BatchNorm ── ReLU ── GAP ──┐
  └──────────────────────────────────────────────────────┘
  ┌─ Dense(16) ── ReLU ── Dense(8) ── Softmax ──┐
  │ {DROWSY, RELAXED, CALM, ATTENTIVE,          │
  │  FOCUSED, DEEP_FOCUS, HYPERFOCUS, NOISE}    │
  └──────────────────────────────────────────────┘

大小: ~4KB (INT8 量化)
推理: <50ms (ESP32-S3 向量扩展 + CMSIS-NN)
RAM:  ~8KB 张量竞技场

模型更新: Phase 1 线上收集数据 → Phase 2 云端重新训练 → OTA 推送新模型
```

### 6.2 云端 AI（个性化训练管道）

```
Phase 0: 无云 → 启发式分类器 (θ/β 比 + α 功率 + δ/β 比)

Phase 1: 数据收集管道
  每天启动时: WiFi 连接 → 上传前一天所有 EEG 数据 (夜间充电)
  → AWS S3 / GCP Cloud Storage → 标注 (用户按下"专注"或"放松"键时的数据即标注)
  
Phase 2: 个性化训练
  基础模型: 在 10,000+ 小时公开 EEG 数据上预训练
  微调: 每个用户 10+ 小时个人数据做迁移学习 → 专属 FocusDetector
  量化: INT8 → OTA 推送到设备

Phase 3: AI 教练 (长期分析)
  AI 发现你的模式:
    ✅ "你周二上午 9 点注意力最高 (+23% over baseline)"
    ✅ "你下午 3 点后专注力持续下降 (-18%)，建议下午做机械性工作"
    ✅ "你用了 3 个月后，平均专注度提升 23%"
    ✅ "你的冥想状态下 α 功率从 8dB 提升到 14dB"
    ✅ "你星期五下午最容易进入深度专注 (>85 分，持续时间 1.5h)"
```

### 6.3 闭环规则引擎

```
输入: 专注度分数 S_focus [0,100]，放松度 S_relax [0,100]
输出: 刺激电流 I_mA，刺激模式 mode

规则 (Phase 1):
  if S_focus > 70:
    I = 0.5 + (S_focus - 70) × 0.025 mA   // 0.5 ~ 1.25 mA
    mode = tDCS_ANODAL (F3 阳极, Fp2 阴极) // 提升前额叶活动
  elif S_relax > 70:
    I = 1.0 mA
    mode = tACS_10Hz                        // α 节律同步诱导放松
  elif S_focus < 40:
    I = 0.5 mA
    mode = tDCS_ANODAL                      // 微量支持
  else:
    I = 0
    mode = IDLE                             // 纯监测

AI 自适应 (Phase 2+):
  不是固定阈值。AI 从你过去的反馈中学习:
    → 你的"专注阈值"可能不是 70，而是 65
    → 你的"最有效 tDCS 电流"可能不是 1.25mA，而是 0.8mA
    → 你的"退步时间"可能是下午 3 点而不是 2 点
    → 规则引擎的每个参数都可以被个人化模型覆盖
```

### 6.4 AI 叙事路线图

| 月份 | 设备端 AI | 云端 AI | 用户感知 |
|------|-----------|---------|---------|
| 1-3 | 启发式检测 | 数据收集管道 | "有趣，能测出我走神了" |

## 7. 固件架构

### nRF5340（传感器域，Zephyr RTOS）

```
main.c:
  init_ble()                    // BLE 5.4 广播 + GATT 服务
  init_ads1299_spi(250Hz)       // SPI DMA, 4ch × 24bit
  init_dac_spi()                // DAC8562, 10MHz
  init_uart()                   // → ESP32-S3, 115200
  init_safety_timer()           // 2Hz 安全轮询

Thread 1 (DRDY ISR @250Hz):    ADS1299 SPI DMA → 环形缓冲
Thread 2 (1Hz):                打包 4ch 数据 → UART → ESP32
Thread 3 (UART Rx):             解析 ESP 闭环命令 → DAC 控制
Thread 4 (BLE Notify @1Hz):    脑状态 → 手机 App
Thread 5 (Safety @2Hz):        阻抗检测 + 过流保护 + 超时监控
```

### ESP32-S3（AI 域，ESP-IDF）

```
Task 1 (Core 1, 1Hz):
  UART Rx → 拆包 4ch 数据
  → FFT (CMSIS-DSP, 512点) → 20 频带功率
  → TFLite 推理 → FocusOutput
  → 闭环规则 → tDCS 参数
  → UART Tx → nRF5340
  → Light Sleep (~950ms, 极低功耗)

Task 2 (Core 0, 每天一次):
  夜间充电时: WiFi 上传昨天数据 → OTA 检查

Task 3 (Core 0):
  日志缓冲 → SPI Flash 写入
```

## 12. 脑电数据资产

### 为什么脑数据是最有价值的训练数据

```
层级           数据价值          谁在买
──────        ─────────        ──────────
用户行为      $0.01/h          Google/Facebook (已有)
用户搜索      $0.05/h          Google (已有)
用户购买      $0.10/h          Amazon (已有)
用户位置      $0.20/h          Uber/Lyft (已有，限精度)
用户语音      $0.05/h          Apple/Amazon/Google (已有)
用户大脑      $0.50-5.00/h     ⚡ 目前不存在
```

**为什么脑数据价值高两个数量级？**

1. **行为数据的源头** — 所有点击、购买、搜索、滑动之前，大脑已经先做出决策
2. **不能被伪造** — 用户可以点错广告，但大脑不会"假装专注"
3. **通用人工智能的最后一块拼图** — ChatGPT 已经吞噬了人类所有公开文本。下一步是让 AI 理解**人的注意力、情感、决策机制**——而这只有脑数据能提供
4. **神经经济学的应用** — advertisers, product designers, and AI trainers all want to understand human cognition at the source

### 数据资产化路线

```
Phase 1 (2026 Q4):
  数据量: 50,000h (100 用户 × 500h/年)
  数据类型: 脑状态 (专注/放松)，环境上下文 (时间/地点)
  价值: 内部训练 AI + 基础趋势报告 (对内)
  
Phase 2 (2027 Q2+):
  数据量: 500,000h (1,000 用户)
  数据类型: 脑状态 + 刺激效果 + 环境上下文 + 自报告
  价值: 匿名、归类、去标识化后的训练数据
  买家: AI 公司在训练"注意力预测"和"人机交互优化"模型
  
Phase 3 (2027 Q4+):
  数据量: 5,000,000h+ (10,000+ 用户)
  数据类型: 脑状态 + 刺激 + 环境 + 认知评估 + 多模态融合
  价值: 每个用户大脑的"数字孪生"——预训练模型 + 个人微调数据
  买家: 可以从我们这里购买"Cognitive Attention API"而不需要自己去收集脑数据
```

### 数据商业模式

```
模式 A: 内部训练（免费层）
  → 只用用户自己的数据训练他个人的模型
  → 用户完全控制，免费

模式 B: 匿名联合训练（opt-in, 默认开）
  → 去标识化后供全局 AI 模型联合训练
  → 提升所有人的模型准确性
  → 用户获得更好的 AI 免费功能

模式 C: 数据授权（opt-in, 需要明确同意）
  → 匿名脑数据出售给 B2B 方 (AI 公司, 研究机构)
  → 用户分享数据收入 (50%)
  → 每用户年收入 $20-50

每年如果 NeuroResonator 有 10,000 用户在模式 C:
  → 总数据价值 = 10,000 用户 × 1,000 小时/年 × $0.5/h = $5M/yr
  → 用户分得 $2.5M → 用户赚回设备成本
  → NeuroResonator 分得 $2.5M → 每年从数据授权中获得超过设备销售的利润

长期目标:
  → 用户免费使用设备（数据收入覆盖硬件成本）
  → NeuroResonator 实际上是一个免费硬件 + 高利润数据平台
  → 类似 OpenAI 从免费 API 到付费模型的转变，但这里的"数据"是人的大脑状态
```

### 风险与缓解

| 风险 | 缓解 |
|------|------|
| 隐私担忧 | 所有数据默认本地设备，仅经明确 opt-in 后上传。匿名化 + 去标识化。 |
| 伦理争议 (AI 读取思维) | 产品定位"健康辅助设备"，非"思维读取器"。AI 只分析状态，不分析内容。 |
| 用户退出门槛 | AI 模型属于用户，离开 NeuroResonator 时带走自己的模型 |
| 监管打击 | 私人岛屿 + 美国消费级法律约束 → 目前没有针对"脑健康数据"的具体法规 |
